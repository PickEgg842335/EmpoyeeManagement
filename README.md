# EmpoyeeManagement
This homework is design the Empoyee Management system by nodejs
## Function
1. 實作前端頁面，提供首頁，新增員工資料介面，查詢所有員工/個別員工資料介面
2. 實作後端寫檔操作，讀檔查詢操作
